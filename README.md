### Backend

- Node.js
- API: REST
- Database: MongoDB



## Run Project

### Prerequisites

- install node (incl. npm)
- (install docker)

### Get started

1. pull repository
2. cd into ./backend
3. `npm install`
4. `npm start`
5. cd into ./frontend
6. `npm install`
7. `npm start`

### Testing

1. pull repository
2. cd into ./backend
3. `npm test`
4. cd into ./frontend
5. `npm test`
